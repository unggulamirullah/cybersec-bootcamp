# Lab Day 2 - Git & GitHub

## 🎯 Objective

Membuat repository Git dan menghubungkannya dengan GitHub.

---

## Environment

- Git
- GitHub
- WSL Ubuntu

---

## Langkah Praktikum

### Inisialisasi Repository

```bash
git init
```

### Cek Status

```bash
git status
```

### Menambahkan File

```bash
git add .
```

### Commit

```bash
git commit -m "Initial Commit"
```

### Push ke GitHub

```bash
git push
```
