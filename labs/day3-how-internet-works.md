# Lab Day 3 - How Internet Works

## 🎯 Objective

Mengamati proses komunikasi jaringan menggunakan command Linux.

---

## Tools

- nslookup
- ping
- traceroute
- curl

---

## Langkah Praktikum

### DNS Lookup

```bash
nslookup google.com
```

### Ping

```bash
ping google.com
```

### Traceroute

```bash
traceroute google.com
```

### HTTP Header

```bash
curl -I https://google.com
```
