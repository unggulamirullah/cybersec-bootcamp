# Cybersecurity Glossary

## DNS

Domain Name System. Mengubah nama domain menjadi IP Address.

---

## IP Address

Alamat unik yang digunakan perangkat dalam jaringan.

---

## TCP

Transmission Control Protocol. Protokol komunikasi yang memastikan data sampai dengan benar.

---

## HTTP

Hypertext Transfer Protocol.

---

## HTTPS

HTTP yang diamankan menggunakan TLS.

---

## SSH

Secure Shell. Digunakan untuk mengakses sistem secara aman dan autentikasi GitHub.

---

## Git

Version Control System untuk melacak perubahan file.

---

## Repository

Tempat penyimpanan project yang dikelola menggunakan Git.
